가설 검정

1. 추론 통계의 개요

- 추론통계란? 모집단에서 샘플링한 표본을 가지고 모집단의 트것ㅇ을 추론하고 그 결과가 신뢰성이 있는지 검정하는 것
- 표본은 모집단의 특성을 대변해야 함.

추론 통계시 주의할 점
- 표본집단은 모집단을 대표할 수 있어야 한다.
- 표본의 확률분포는 어떠한가? > 어떤 분포냐에 따라 추정을 위한 기법이 달라지기 때문에 중요함.
  다만, 표본의 수가 많아질 수록 정규분포에 근접하게 된다.
- 추정된 결과는 신뢰성이 있는가? // p-value < 유의수준
  추정된 결과를 활용할 수 있는지를 결정하는 요소이기 때문에 중요함.

2. 가설 검정의 절차

가설 설정 > 유의수준 설정 > 검정통계량 산출 > 기각/채택


# 귀무가설(=영가설=H0) : 일반적으로 맞다고 가정하는 가설을 말한다.
# 차이가 없다, 영향력이 없다, 연관성이 없다, 효과가 없다.
# 대립가설(=H1) : 새롭게 맞다고 증명하려는 가설을
# 차이가 있다, 영향력이 있다, 연관성이 있다, 효과가 있다.

검정방법

# 양측검정 : 귀무가설 기각역이 양쪽에 있는 검정을 말한다.(크거나 작다 = 아니다 없다)
# 단측검정 : 한쪽에만 치우친 값(크다, 작다)

# 신뢰수준 : 가설을 검정할 때 얼마나 정확하게 검정할 것인지 결정하는 수준 (일반적으로 95%)
# 유의수준 : 가설을 검정할 때 이 정도까지 벗어나면 귀무가설이 오류인걸 인정하는 수준
# 유의수준 = 1-신뢰수준
# 유의수준 = ∑ 기각역
# 기각역 : 귀무가설을 기각하는 영역 양측검정인 경우 기각역은 유의수준/2 단측검정인 경우 유의수준과 같음.
# 신뢰구간 : 신뢰수준에 포함되는 x값 구간
# 임계치 : 신뢰구간에서 기각역으로 넘어가는 기준이 되는 x값
# 검정통계량 : 가설을 검정하기 위한 기준으로 사용하는 값
			검정통계량이 확률분포 상 어디에 위치하는지에 따라 귀무가설을 기각하거나 기각하지 않는다.
# 유의확률(p-value) : 자유도를 고려했을 때 검정통계량에 대한 확률을 말함(귀무가설이 신뢰구간을 벗어나는 확률)
